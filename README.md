# 📈 Magnificent 7 Stock Market Analysis Dashboard (Power BI)

## 📘 Project Overview

This project analyzes the historical stock market performance of the **Magnificent 7 technology companies** using **Python** and **Power BI**.

The analysis focuses on stock price trends, cumulative returns, volatility, trading activity, risk vs return comparison, and investment behavior across major technology companies from **2016 to 2026**.

The project combines **Python-based financial analysis** with **interactive Power BI dashboards** to generate business insights and visualize market trends effectively.

The dataset contains historical stock market data collected from **Yahoo Finance** using the **yFinance API**.

---

## 📊 Dataset Schema

| **Column Name** | **Description**                |
| --------------- | ------------------------------ |
| Date            | Trading date                   |
| Ticker          | Stock symbol of the company    |
| Open            | Opening stock price            |
| High            | Highest stock price of the day |
| Low             | Lowest stock price of the day  |
| Close           | Closing stock price            |
| Adj Close       | Adjusted closing stock price   |
| Volume          | Total number of shares traded  |
